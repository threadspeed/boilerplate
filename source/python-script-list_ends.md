## python list ends

Python example: list ends

```python
x = []
n = int(input("Enter number of elements:"))
for each in range(0, n):
    each = int(input("Enter a number:"))
    x.append(each)
first_and_last = [each for each in (x[0], x[-1])]
print(first_and_last)

```
