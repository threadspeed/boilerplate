## python Namespiral

Python mysql example: Namespiral

```python
import turtle
t = turtle.Pen()
name=turtle.textinput("Name","What is your name?")
color=["red","orange","yellow","green","blue","purple"]
t.speed(999999999)
for x in range(1000):
    turtle.bgcolor("black")
    t.pencolor(color[x%6])
    t.penup()
    t.forward(x)
    t.pendown
    t.write(name,font="Arial")
    t.left(61)



```
