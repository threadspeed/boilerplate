## python arguments

Python example: arguments

```python
### Позиционные аргументы vs. аргументы по умолчанию

'''
Задача: написать функцию, которая будет здороваться с пользователем.
Функция должна принимать два аргумента: имя пользователя и приветствие.
'''

'''
Первый вариант функции: оба аргумента - позиционные.
Если хотим с кем-то поздороваться, должны указывать и того, с кем здороваемся, и то, как здороваемся.
Очевидный минус: мы чаще всего всем говорим "привет", поэтому придется постоянно прописывать это "привет" при вызове функции.
Когда-нибудь точно надоест, не так ли?
'''

def hello(name, phrase):
    print(phrase + ", " + name + "!")

hello("Петя", "Привет")
hello("Маша", "Привет")
hello("Алексей Юрьевич", "Здравствуйте")

'''
Второй вариант: давайте всем по умолчанию говорить привет.
Тогда у нас один аргумент позиционный (имя), а другой - по умолчанию (приветствие).
Если нам захочется поздороваться как-то иначе, мы тогда вызовем функцию от двух аргументов, прописав то приветствие, которое хотим употребить.
'''

def hello_smart(name, phrase = "Привет"):
    print(phrase + ", " + name + "!")

hello_smart("Петя")
hello_smart("Маша")
hello_smart("Алексей Юрьевич", "Здравствуйте")

# главное не забывать о том, что там у нас за аргумент по умолчанию...
hello_smart("Алексей Юрьевич") 

'''
Аргументов по умолчанию может быть больше одного.
Давайте модифицируем нашу функцию, добавив в нее вторую фразу.
'''

def start_conversation(name, first_phrase = "Привет", second_phrase = "Как дела?"):
    print(first_phrase + ", " + name + "!", second_phrase)

start_conversation("Маша")
start_conversation("Петя", "Эй")
start_conversation("Алексей Юрьевич", "Здравствуйте", "У меня вопрос по домашнему заданию...")

'''
Не забывайте, что аргументы функции надо отдавать в определенном порядке.
Что в таком случае делать, если хотите при вызове функции изменить какой-то из аргументов по умолчанию (не первый)? Указывать, чему он должен быть равен.
'''

start_conversation("Петя", second_phrase = "Ты сделал домашку?")

'''
Замечание: на самом деле, лучше располагать аргументы по умолчанию в таком порядке, чтобы те из них, которые меняются реже остальных, оказывались в конце.
Почему это важно? Так мы экономим свои же усилия.

Возвращаясь к примеру выше, заметим, что приветствие меняется не так часто, как то, что мы говорим после. 
Поэтому лучше было бы при определении функции располагать аргументы в другом порядке: name, second_phrase, first_phrase.
'''

'''
Позиционных аргументов, разумеется, тоже может быть больше одного. 
'''

def hello_two(name1, name2, phrase = "Привет"):
    print(phrase + ", " + name1 + " и " + name2 + "!")
    
hello_two("Петя", "Маша")
hello_two("Алексей Юрьевич", "Татьяна Владимировна", "Здравствуйте")    

# Только не забывайте, что позиционные аргументы - обязательные. Пропускать их при вызове функции нельзя.
# Вот так делать нельзя:
hello_two("Алексей Юрьевич", "Здравствуйте")  


```
